## 📋 Sobre o Projeto

Este é meu portfólio pessoal, desenvolvido para apresentar minhas habilidades, projetos e experiências como desenvolvedor web. O site foi criado com foco em design moderno, performance e responsividade, utilizando apenas tecnologias front-end fundamentais.

### ✨ Características Principais

- 🎨 **Design Moderno**: Interface limpa e profissional com gradientes e efeitos visuais
- 📱 **Totalmente Responsivo**: Adaptado para todos os dispositivos (mobile, tablet, desktop)
- ⚡ **Performance Otimizada**: Carregamento rápido e animações suaves
- 🌙 **Tema Dark**: Design elegante com esquema de cores escuro
- 🎭 **Animações CSS**: Efeitos de hover, transições e animações de entrada
- 📧 **Formulário de Contato**: Integração com Formspree para recebimento de mensagens

## 🚀 Como Executar

### Pré-requisitos
- Navegador web moderno
- Servidor local (opcional, mas recomendado)

### Instalação

1. **Clone o repositório**
   ```bash
   git clone https://github.com/ViniciusSavianDeArruda/portfolio.git
   ```

2. **Navegue até o diretório**
   ```bash
   cd portfolio
   ```

3. **Execute localmente**
   
   **Opção 1: Live Server (VS Code)**
   - Instale a extensão Live Server
   - Clique com o botão direito em `index.html`
   - Selecione "Open with Live Server"
   
   **Opção 2: Python**
   ```bash
   python -m http.server 8000
   ```
   
   **Opção 3: Node.js**
   ```bash
   npx serve .
   ```

4. **Acesse no navegador**
   ```
   http://localhost:8000
   ```
