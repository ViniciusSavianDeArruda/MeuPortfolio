## 📋 Sobre o Projeto

Este portfólio foi desenvolvido com foco em **performance**, **acessibilidade** e **experiência do usuário**. Apresenta minhas habilidades como desenvolvedor web front-end, projetos realizados e formas de contato profissional.

### 🎯 Objetivos
- Demonstrar competências técnicas em desenvolvimento web
- Apresentar projetos de forma visual e interativa
- Facilitar contato profissional
- Servir como cartão de visitas digital

## 🔧 Customização

### **Cores**
Edite as variáveis CSS em `assets/styles/base/cores.css`:
```css
:root {
  --cor-primary: rgb(0, 173, 111);
  --cor-secondary: #04FF6E;
  /* ... outras cores */
}
```

### **Conteúdo**
- **Textos**: Edite diretamente no `index.html`
- **Imagens**: Substitua em `assets/images/`
- **Projetos**: Modifique a seção projetos no HTML
- **Skills**: Atualize ícones e nomes das tecnologias

### **Estilos**
- **Layout**: `assets/styles/components/`
- **Cores**: `assets/styles/base/cores.css`
- **Tipografia**: `assets/styles/base/tipografia.css`
- **Responsividade**: Breakpoints em cada componente
